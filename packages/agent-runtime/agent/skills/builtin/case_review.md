# Case Review Skill

## Purpose

Review case material and produce a lightweight evidence-based assessment.

## Inputs

- `query_or_topic`: case question or review focus.
- `business_area`: optional domain filter.
- `document_ids`: optional case material scope.
- `extra_requirements`: optional review criteria.

## Evidence Rules

- Keep the review tied to retrieved case evidence.
- Mark assumptions clearly.
- Identify evidence gaps.
- Do not include sensitive long-form source excerpts.

## Output

- Case summary.
- Relevant facts.
- Issues and risks.
- Suggested next checks.
- Citations and warnings.
