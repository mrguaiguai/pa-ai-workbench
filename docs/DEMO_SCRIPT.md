# PA AI Workbench Demo Script

This file is a placeholder for the MVP demo script. Task F1 will expand it into a complete walkthrough after the backend, agent runtime, knowledge engine, and frontend flows are implemented.

## Draft Flow

1. Open the workbench home page and confirm backend status.
2. Upload or select mock documents in the library.
3. Ask a knowledge question and review citations.
4. Run a policy analysis workflow and inspect warnings or insufficient-evidence messages.
5. Search the wiki cache and open a page.
6. Review generated history.

## Demo Safety

- Use mock data or sanitized sample documents only.
- Do not use real department materials, API keys, local databases, upload artifacts, or logs.
- Keep mock sources clearly marked as `source=mock`.
