# Phase 5 Frontend Terminology Glossary

本文件记录 P5-E1 中文化中处理的英文术语、建议中文显示和含义。原则是：普通用户界面优先显示中文；专有名词、品牌名、API 字段、调试字段可以在必要位置保留，但应只出现在品牌、配置或调试上下文中。

| English term | 中文显示 | 含义 | 显示原则 |
| --- | --- | --- | --- |
| PA AI Workbench | PA 智能工作台 | 本产品名称。 | 品牌区可保留英文全称，普通标题用中文。 |
| RAG | RAG 检索 | 检索增强生成能力，先检索证据再生成回答。 | 可保留 RAG，但需要搭配中文说明。 |
| Wiki | Wiki 知识库 | 可沉淀、发布和检索的专题知识页。 | 作为产品模块名可保留。 |
| WeKnora | WeKnora | 提供知识库、检索、Wiki、Agent 等平台能力。 | 品牌名保留。 |
| Agent | Agent 分析流 | 内置智能分析工作流。 | 普通页面显示为“Agent 分析流”。 |
| Workflow | 分析流 | 知识问答、政策分析、案例复盘等任务流。 | 普通页面用中文。 |
| Workspace | 工作区 | 进入资料库、分析、Wiki、历史等入口的区域。 | 普通页面用中文。 |
| Overview | 总览 | 首页概览。 | 顶部页签用中文。 |
| Library | 资料管理 / 资料库 | 上传和查看资料的页面。 | 导航和标题用中文。 |
| Analysis | 分析 | 智能分析页面。 | 顶部页签用中文。 |
| Retrieve Debug | 检索调试 | RAG 调试页面。 | 顶部页签用中文。 |
| History | 历史 | 生成历史页面。 | 顶部页签用中文。 |
| Chat Model | 对话模型 | 生成回答的模型配置和运行状态。 | 首页状态卡用中文。 |
| Embedding | 向量模型 / 向量化 | 将文本转成向量用于检索。 | 普通状态用中文；字段名可在调试区保留。 |
| RAG Pipeline | RAG 检索链路 | 从知识库检索 evidence 的链路状态。 | 首页状态卡用中文。 |
| Capability | 能力边界 | 当前后端支持、部分支持、不支持的能力摘要。 | 首页状态卡用中文。 |
| configured | 已配置 | 必要配置已提供。 | 状态显示用中文。 |
| missing config | 配置缺失 | 必要配置未提供。 | 状态显示用中文。 |
| unavailable | 不可用 | 能力或服务当前不可用。 | 状态显示用中文。 |
| supported | 支持 | 后端明确支持该能力。 | 状态显示用中文。 |
| partial | 部分支持 | 能力只覆盖一部分或存在降级。 | 状态显示用中文。 |
| unsupported | 不支持 | 后端不支持该能力。 | 状态显示用中文。 |
| ready | 准备就绪 | 可开始操作或状态正常。 | 状态显示用中文。 |
| mock | 模拟模式 / 模拟证据 | 非真实能力或模拟数据。 | 普通页面用中文；API 值保持原样。 |
| fallback | 回退 | 主能力不可用时使用替代路径。 | 普通页面用中文。 |
| real ready | 真实能力就绪 | 当前链路不依赖模拟模式。 | 首页状态用中文。 |
| mock fallback | 模拟模式回退 | 当前链路回退到模拟能力。 | 首页状态用中文。 |
| fail closed | 失败关闭 | 能力不可证明时阻止继续冒充可用。 | 首页状态用中文。 |
| Upload | 上传 | 上传资料入口。 | 普通页面用中文。 |
| Documents | 资料列表 | 已上传资料列表。 | 普通页面用中文。 |
| Chunks | 分块 | 文档切分后的文本块。 | 普通页面用中文。 |
| Events | 事件 | 文档处理过程中的近期事件。 | 普通页面用中文。 |
| uploaded | 已上传 | 资料已上传但可能未处理完。 | 下拉选项用中文，API 值保持原样。 |
| processing | 处理中 | 资料正在解析、分块、向量化或索引。 | 下拉选项用中文。 |
| indexed | 已索引 | 资料已进入检索索引。 | 下拉选项用中文。 |
| failed | 失败 | 处理失败。 | 下拉选项用中文。 |
| extracted | 本地抽取 | 使用本地抽取型后端。 | 下拉选项用中文。 |

| chars | 字符 | 文本字符数量。 | 普通页面用中文。 |

| Source | 检索来源 | 文档、Wiki 或全部来源。 | 普通页面用中文。 |

| KB ID | 知识库 ID | WeKnora 知识库标识。 | 调试页用中文。 |

| Run | 运行 | 触发调试检索。 | 普通页面用中文。 |
| Reset | 重置 | 清空调试表单和结果。 | 普通页面用中文。 |
| hits | 命中 | 检索结果数量。 | 普通页面用中文。 |
