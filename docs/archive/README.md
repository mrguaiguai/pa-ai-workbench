# Historical documentation archive

Archived files are preserved evidence and specifications from completed or
superseded stages:

- `legacy-product/` — Day-1 and PA v0.1–v0.5 product/development specs;
- `phase3/`, `phase4/`, `phase5/` — completed phase records;
- `weknora-first/` — five-day WeKnora-first sprint records;
- `wnx/` — WeKnora Native Expansion records;
- `wnfc/` — WeKnora Native Full Completion records;
- `wnid/` — WeKnora Native Intelligent Dialogue records.

These files retain their original claims and narrative. Relative links are
maintained for auditability, but archived PASS statements do not validate the
current checkout.
